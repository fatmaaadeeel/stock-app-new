# 📊 Stock Market Dashboard

## 🔹 Overview

This is a Streamlit web app for analyzing stock market data.

## 🔹 Features

* Enter stock symbol
* View current price
* Show charts
* Display historical data
* Moving average

## 🔹 Run the project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
